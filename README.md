# Tromp

Ett React-projekt byggt med Vite och TypeScript.

## Kom igång

### Installation

Installera beroenden:

```bash
npm install
```

### Utveckling

Starta utvecklingsservern:

```bash
npm run dev
```

Öppna [http://localhost:5173](http://localhost:5173) i din webbläsare för att se applikationen.

### Bygga för produktion

Bygg projektet för produktion:

```bash
npm run build
```

Förhandsgranska produktionsbyggen:

```bash
npm run preview
```

## Teknologier

- **React** - UI-bibliotek
- **Vite** - Byggverktyg och utvecklingsserver
- **TypeScript** - Typat JavaScript
- **ESLint** - Kodkvalitet och linting
